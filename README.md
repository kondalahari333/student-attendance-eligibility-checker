# student-attendance-eligibility-checker
##
this java program calculates a student's attendance percentage and determines whether the student is eligible to attend the examination.
-->As per college rules,a minimum of 75% attendance is required.
##features
*Takes user input for attendance details.
*calculates attendance percentage accurately.
*Display exam eligibility status clearly.
*Handles invalid inputs gracefully.
##concepts used
*java
*Conditional statements
*Arithmetic calculations
#input
*Total working days
*Attended days
#sample output
Enter total working days:
120
Enter attended days:
90
Attendance percentage:75.0%
Eligible for exam
